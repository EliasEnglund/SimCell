# Resources And Economy

Resources should connect metabolism, protein synthesis, membrane transport, and DNA research.

## Current Core Resources

- ATP: energy.
- NADH: electrons/redox balance.
- Amino Acids: protein construction.
- N: nitrogen resource.
- DNA Points: research currency.

## Economy Rules

Enzyme reactions can:

- produce ATP,
- consume ATP,
- produce NADH,
- consume NADH,
- consume nitrogen,
- release CO2,
- convert target molecules into resources.

The player should eventually need redox balance: NADH generated in one pathway must be consumed elsewhere or managed by unlocks.

## UI Requirements

The top resource bar should show compact icons and numbers. Hovering a resource should show:

- icon,
- name,
- current number,
- rate or explanation when available.
