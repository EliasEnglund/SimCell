# DNA Tech Tree

DNA research is the long-term unlock system.

## Visual Concept

The tree starts from a root node near the bottom and branches upward like a tree of life. Connections should look like DNA strands. As research progresses, the strand between parent and child lights up proportionally.

Nodes are circular icons. Locked nodes are gray. Available nodes glow subtly. Completed nodes are bright and readable.

## Interaction

- The workspace can be panned by click-drag.
- The camera should center on the bottom/root node the first time the view opens.
- Research spends DNA points.
- Progress should be visible on both node and DNA-strand connection.

## Chemistry Tier Direction

- Tier 1: Nitrogen chemistry.
- Tier 2: Phosphorus chemistry.
- Tier 3: Sulfur chemistry.
- Tier 4: Metals and advanced catalysts.

Each tier should unlock new reactions or more efficient versions of existing reactions, not just passive numbers.
