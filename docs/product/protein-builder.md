# Protein Builder

The protein builder shows what the cell is currently synthesizing.

## Purpose

Enzyme and transporter blueprints should become physical proteins through a build queue. This makes amino acids/protein points meaningful and creates a capacity bottleneck.

## Screen Direction

The view should show:

- active ribosome lanes,
- current protein being synthesized,
- progress bars,
- queued templates,
- completed proteins grouped by type,
- synthesis speed and capacity.

## Gameplay Role

Protein building should create decisions:

- build enzymes to expand metabolism,
- build transporters to change source intake,
- build ribosomes to increase production capacity,
- replace unstable enzymes later.
