# Game Loop

The player starts with a simple cell and limited chemistry. The cell imports raw molecules, transforms them through metabolism, converts target molecules into resources, and uses those resources to build better cellular machinery.

## Loop Stages

1. Source molecules exist outside the cell.
2. Membrane transporters import selected molecules.
3. Imported molecules appear as pools on the metabolic board.
4. The player designs enzymes from molecule structures.
5. Enzymes create reaction routes and product pools.
6. Flux particles show conversion rates.
7. Target molecules become resources.
8. Resources build enzymes, transporters, and research.
9. DNA research unlocks new enzyme classes and molecule elements.

## Early Game

The first loop should be small:

- import glucose,
- split or modify it,
- introduce nitrogen,
- make amino acid target,
- convert amino acid target into amino acid resource,
- spend amino acids on more enzymes or transporters.

## Resource Pressures

The long-term loop should require balancing:

- ATP production and ATP cost,
- NADH production and NADH consumption,
- source molecule availability,
- enzyme stability and replacement,
- protein synthesis capacity,
- membrane import/export capacity.
