# Art Direction

SimCell should look like a polished biological engineering interface.

## Visual Identity

- Dark teal/black biological background.
- Cyan-green glow used for important structure and selected states.
- Warm cytoplasm colors inside the cell.
- Cooler blue/teal colors outside the cell.
- Rounded but not overly soft UI panels.
- Scientific, clean, high-contrast typography.

## Avoid

- muddy gray panels,
- overly transparent assets,
- unreadable glow,
- single-hue screens,
- decorative effects that obscure simulation data,
- random style changes between views.

## Reference Direction

Molecules should feel like glossy balls with:

- thin black outer stroke,
- subtle inner rim close to the atom body color,
- clear gradient from lower-left darker area toward upper-right highlight,
- visible bonds with enough spacing between atoms,
- double bonds that do not overlap.

The membrane should feel layered and organic, but it must be readable and performant.
