# Gameplay Review Checklist

Use this after mechanics changes.

## Questions

- What can the player do now that they could not do before?
- Is the action clearly discoverable?
- Does the game show feedback immediately?
- Does the action change resources, rates, layout, or unlocks?
- Is there a reason to make a different choice?
- Can the player recover from a bad choice?

## First Loop Test

1. Start from title screen.
2. Enter cell view.
3. Go to membrane view.
4. Select an outside molecule.
5. Build/import with transporter.
6. Go to metabolism.
7. Select glucose.
8. Open enzyme designer.
9. Create an enzyme blueprint.
10. Wait for protein build.
11. Confirm flux appears.
12. Confirm product appears or target resource is produced.

Any failure in this loop is high priority.
