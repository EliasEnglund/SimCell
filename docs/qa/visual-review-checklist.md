# Visual Review Checklist

Use this after UI or art changes.

## General

- The screen opens without errors.
- Primary action is visually obvious.
- Text is readable at default window size.
- No text overlaps buttons, panels, or icons.
- No UI element clips outside its intended bounds.
- Hover and selection states are distinct.

## Metabolism

- Grid belongs to the world and pans/zooms with it.
- Molecule marbles snap to grid centers.
- Arrows are orthogonal.
- Arrows stop outside marbles.
- Input and output arrows avoid the same node side.
- Flux particles are visible but not noisy.
- Hover popups show correct molecule/reaction detail.

## Membrane

- Outside and inside regions are visually distinct.
- Transporters sit on the membrane, not floating above or below it.
- Right list hover highlights matching center molecules.
- Build importer button appears only with valid selection.
- Side panels match the rest of the UI.

## Cell View

- Cell remains readable at zoom extremes.
- Background particles are behind the cell.
- Movement and flagellum animation match input.
- Environment objects do not clutter the screen.
