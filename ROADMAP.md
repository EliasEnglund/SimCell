# Roadmap

This roadmap is a working development path, not a promise of final scope.

## Current Phase: Prototype The Core Loop

The current priority is to make the enzyme/metabolism/membrane/resource loop understandable and fun.

Focus:

- stable metabolic board interactions,
- clear molecule and reaction visuals,
- enzyme creation that reliably affects simulation,
- membrane importer creation and layout,
- resource feedback for amino acids, ATP, NADH, nitrogen, and DNA points,
- clean visual language across views.

## Next Milestone: Playable Biochemistry Slice

Requirements:

- Start in cell view from title screen.
- Import glucose through a visible transporter.
- See glucose appear on the metabolism board.
- Create at least one enzyme pathway.
- Watch flux particles move along routes.
- Produce amino acid resource from a target molecule.
- Build at least one new enzyme or transporter from that resource.
- Spend DNA points on one tech-tree unlock.

## Later Milestones

Membrane and environment:

- More source molecules outside the cell.
- Inspectable transporter classes.
- Import/export cost differences.
- Environmental scarcity and abundance.

Protein synthesis:

- Ribosome lanes.
- Protein build priorities.
- Enzyme stability and replacement.
- Transporter/protein categories.

DNA research:

- Tiered enzyme unlocks.
- Nitrogen as tier 1 chemistry.
- Phosphorus as tier 2 chemistry.
- Sulfur as tier 3 chemistry.
- Metals as tier 4 catalysts for efficient enzymes.

Polish:

- Better audio control.
- Save/load.
- Performance budget per view.
- Screenshot-based visual QA.
- Steam-ready desktop packaging.
